A mod that prints [b]World Age[/b], [b]Date Time[/b], and [b]Weather[/b] every 10 seconds in the server console.

This tool is created mainly for [url=https://mbgplayground.xyz/projectzomboid]MBG Project Zomboid[/url] server metrics.

Github - [url=https://github.com/mightybeegaming/mbg-worldinfo]MBG World Info[/url]

[hr][/hr]
[b]Snow[/b]
- Snow+ = Blizzard
- Snow = Normal Snow

[b]Rain[/b]
- Rain++ = Heavy Rain
- Rain+ = Normal Rain
- Rain = Light Rain

[b]Cloud[/b]
- Cloud+ = Overcast
- Cloud = Cloudy

[b]Fog[/b]
- Fog+ = Heavy Fog
- Fog = Foggy

[b]Wind[/b]
- Wind+ = Strong Wind
- Wind = Windy
