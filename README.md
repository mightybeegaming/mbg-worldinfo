# MBG World Info
A Project Zomboid mod that prints **World Age**, **Date Time**, and **Weather** in server console.

This tool is created mainly for [MBG Project Zomboid](https://mbgplayground.xyz/projectzomboid) server metrics.

Steam Workshop - [MBG World Info](https://steamcommunity.com/sharedfiles/filedetails/?id=3676979895)

## Weather Labels
| Weather   | Labels                |
| -         | -                     |
| **Snow**  | Snow+ = Blizzard      |
|           | Snow = Normal Snow    |
| **Rain**  | Rain++ = Heavy Rain   |
|           | Rain+ = Normal Rain   |
|           | Rain = Light Rain     |
| **Cloud** | Cloud+ = Overcast     |
|           | Cloud = Cloudy        |
| **Fog**   | Fog+ = Heavy Fog      |
|           | Fog = Foggy           |
| **Wind**  | Wind+ = Strong Wind   |
|           | Wind = Windy          |
